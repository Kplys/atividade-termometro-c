# Atividade Termômetro - C

Repositório com as resoluções da atividade de nivelamento de C.

## Arquivos
- `1_impressao.c`: Imprime de 1 a N
- `2_fatorial.c`: Cálculo de fatorial
- `3_soma_array.c`: Soma de 5 números num array
- `4_temperatura.c`: Conversão Celsius/Fahrenheit
- `5_contagem_string.c`: Conta caracteres de uma string
- `6_numero_primo.c`: Checa se o número é primo

## Como rodar

Você vai precisar do GCC instalado. Abre o terminal na pasta dos arquivos e roda:

1. Para compilar:
gcc nome_do_arquivo.c -o programa

2. Para executar (Linux/Mac):
./programa

3. Para executar (Windows):
programa.exe
